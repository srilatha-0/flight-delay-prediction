# ✈️ Flight Delay Prediction using Machine Learning

This project predicts whether a flight will be delayed based on various factors such as airline, origin, destination, weather conditions, and time-related features.
It also provides data analysis insights using the dataset.

The application is built with **Streamlit** for an interactive web interface and uses Machine Learning models for prediction.

---

## 🚀 Features

* Predicts flight delays using input parameters
* Interactive web app built with Streamlit
* Data analysis and visualization of flight dataset
* Uses ensemble Machine Learning models
* User-friendly interface

---

## 🧠 Machine Learning Models Used

* Random Forest Classifier 🌳
* Ensemble-based approach for better accuracy

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn (for analysis)
* Streamlit

## 📂 Project Setup Instructions

### 1️⃣ Clone the Repository

```bash id="b7m2u0"
git clone https://github.com/your-username/flight-delay-prediction.git
cd flight-delay-prediction
```

---

### 2️⃣ Download the Dataset

Download the flight dataset from a trusted source.

After downloading:

👉 Move the dataset file into the project folder
👉 Place it inside the **data/** directory

Example structure:

```id="6f4w5j"
flight-delay-prediction/
│
├── data/
│   └── flights.csv
```

---

### 3️⃣ Install Dependencies

```bash id="z1n3pb"
pip install -r requirements.txt
```

---

### 4️⃣ Run the Streamlit Application

```bash id="i1yq5t"
streamlit run app.py
```

---

## 💻 How to Use

1. Launch the Streamlit app
2. Enter flight details (airline, origin, destination, weather, time, etc.)
3. Click **Predict**
4. The model will indicate whether the flight is likely to be delayed
